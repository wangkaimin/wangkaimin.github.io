Spring Cloud 介绍

  1. Spring Cloud是一个集合，是一个开发工具集，集合了众多的开源框架，注意他不是自己造轮子，它吸收了很多优秀的开源框架。
  2. Spring Cloud利用Spring boot的开发便利性，实现了很多功能，例如服务注册，服务发现、负载均衡、数据监控、RESTFull API等等。

  3. 基本囊括了分布式框架所需要的所有功能，在整合过程中呢，最主要的是基于对Netflix开源组件的进一步封装。

  4. 不知道大家知不知道Netflix，在国内喜欢看美剧给他起了一个中文名子，叫耐非，他是美国一家在线视频网站，有点像类似国内的youku、iqiyi之类的网站

  5. 微服务业的翘楚就是Netflix，他是公认的大规模生产级微服务做的最杰出的实践者。

Netflix开源组件已经在他的大规模分布式微服务环境中经过了多年的生产实战验证，所以呢Spring Cloud的很多组件都是基于Netlifx开源组件的进一步封装

  1. 然后呢，我们看一下sc的作用，在我看来呢，最大的作用简化了分布式开发，极大的降低的分布式开发的门槛。

  2. 也正因为如此，我认为掌握Spring Cloud相关组件，怎么用哪些都是其次，我们要更理解分布式的特点，分布式架构的特点，这是最重要的。
